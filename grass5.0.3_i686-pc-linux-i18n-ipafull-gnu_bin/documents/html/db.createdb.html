db.createdb

NAME

db.createdb - create an empty database
(GRASS Database Program)

SYNOPSIS

db.createdb driver=name database=name location=name

Parameters:

driver
Driver name
database
Database name
location
Database location

NOTE

Module does not work with ODBC driver.
