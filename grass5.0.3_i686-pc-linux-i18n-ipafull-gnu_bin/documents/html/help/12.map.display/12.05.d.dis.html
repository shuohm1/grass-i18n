MAP DESIGN, DISPLAY, INPUT AND CAPTURE

Map display, input, and capture functions give users flexibility in designing maps for display to graphics monitors. GRASS macros can be built to run demos making use of these displays. Photos can also be taken of the screen. On some systems, a screen dump can also be made, to capture the displayed image and reformat it for output to a printer.

The display, input, and capture functions can be further categorized as to whether they involve map display, input, or screen capture, and as to whether they involve raster data, vector data, site data, text, or screen graphics.
