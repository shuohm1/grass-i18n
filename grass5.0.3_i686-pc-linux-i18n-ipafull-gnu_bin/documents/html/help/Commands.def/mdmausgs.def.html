m.dmaUSGSread -

m.dmaUSGSread extracts digital terrain elevation data (DTED) from 1/2-inch magnetic tape that was produced by the Defense Mapping Agency (DMA) but bought from the USGS. These two agencies distribute the same data in a slightly different format.
