hsv.rgb.sh -

hsv.rgb.sh is a Bourne shell script program that converts HSV to RGB using r.mapcalc. The Foley and Van Dam algorithm is the basis for this program. Imput must be three raster files - each file supplying the HSV values. Three new raster files are created representing RGB.
