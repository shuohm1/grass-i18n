m.lulc.USGS -

m.lulc.USGS creates raster map layers from a Composite Theme Grid (CTG) file created by the GRASS program m.lulc.read. m.lulc.read extracts the CTG data from an ASCII landuse/landcover (lulc) CTG format file supplied by the USGS.
