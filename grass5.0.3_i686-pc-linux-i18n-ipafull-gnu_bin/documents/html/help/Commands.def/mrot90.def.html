m.rot90 -

m.rot90 is used as a companion program to the DTED and DEM digital elevation tape extraction programs m.dted.extract and m.dmaUSGSread. m.rot09 rotates the contents of the output files generated by these tape extraction programs 90 degrees. Rotation is necessary to compensate for the orientation of the data read from tape. This program can be used in conjuction with the program m.import.ll to convert this rotated data into GRASS raster map layer form.
