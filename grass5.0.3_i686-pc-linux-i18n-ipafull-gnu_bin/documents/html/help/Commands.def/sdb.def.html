s.db -

s.db allows the user to create, manage and query information about site locations (sites) across the landscape in a batch mode or interactively, with input in a command line style. This program represents the marriage of the GRASS environment and the programmer's interface library of the relational database management program RIM distributed by the University of Washington Academic Computing Services.
