d.paint.labels -

d.paint.labels displays the paint label file name in the active display frame on the graphics monitor. This file can be generated by the labels option in the p.labels program, the d.labels program or simply created by the user as an ASCII file (using a text editor like vi) and placed in the appropriate "paint labels" directory under the user's current mapset and location.
