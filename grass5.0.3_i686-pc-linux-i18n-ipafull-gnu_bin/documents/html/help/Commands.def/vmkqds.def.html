v.mkquads -

v.mkquads creates a GRASS vector map layer and/or sites list and/or geographic region definition file for a USGS 7.5-minute quadrangle. There are three types of output available:

1) a vector map of all the full USGS quadrangles that fit within
the boudaries of the current geographic region;

2) a GRASS sites list containing the corner coordinates of each
these quads;

3) GRASS geographic region definition files associated with each
of the quads created.
