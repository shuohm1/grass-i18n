rgb.hsv.sh -

rgb.hsv.sh is a Bourne shell program which converts RGB values to HSV using r.mapcalc. The Foley and Van Dam algorithm is the basis for the program. Input must be three raster files - each file supplying the RGB values. Three new raster files are created representing HSV.
