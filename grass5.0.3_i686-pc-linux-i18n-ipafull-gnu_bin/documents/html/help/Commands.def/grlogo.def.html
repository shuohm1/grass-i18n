grass.logo.sh -

grass.logo.sh is primarily a demonstration of the GRASS d.graph program in a UNIX Bourne shell macro which generates the U.S. Army Corps of Engineers logo. Users are encouraged to generate their own unique logos by writing similar macro shell scripts.
