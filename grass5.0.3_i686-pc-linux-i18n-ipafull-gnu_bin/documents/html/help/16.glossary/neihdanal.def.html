neighborhood anaylsis - In GRASS, an analytic procedure run on a map in its raster format in which a new value is assigned to each cell as a (user-specified) function of the values in the surrounding ("neighborhood") cells. Currently, in GRASS, neighborhoods must be square; their size is specified by the user.
