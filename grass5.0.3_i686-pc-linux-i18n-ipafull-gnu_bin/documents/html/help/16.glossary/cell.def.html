cell - The fundamental unit of information in a raster map. In their raster format, a grid containing rectangular cells of uniform size is superimposed over map data. Each cell is assigned a value (a.k.a. category value) and label (a.k.a. category label) describing the type of data found there. Each cell is also associated with a specific geographic x,y, coordinate location. In GRASS, the user specifies cell dimensions in the x (east-west) and y (north- south) directions, for each map; these may differ. Grid cell size is linked to map scale and resolution. Smaller grid cells enable the assignment of more values to a given area, and are therefore associated with a larger scale and a finer resolution, than are larger grid cells.
